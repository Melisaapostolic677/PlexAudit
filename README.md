## 🎯 What PlexAudit Does

PlexAudit checks your Plex library against the media files on your computer. It helps you find:

- items that exist in Plex but not in your folders
- files in your folders that Plex has not picked up
- name mismatches between Plex and your source media
- library gaps that need a manual fix

It gives you a clear audit so you can see what does not match.

## 📁 What to Select

For best results, pick the right folders:

- Plex library path: the folder or data source Plex uses for your media view
- Source media folder: the folder where your real video files live

Use the exact folder that holds the files you want to check. If you pick a parent folder that contains extra content, the audit may show results you do not want.

## 🧾 What the Results Mean

PlexAudit may show several types of entries:

- **Match**: the file exists in both places
- **Missing in Plex**: the file is on disk but not in Plex
- **Missing on Disk**: Plex shows the item, but the file is gone
- **Name mismatch**: the file names do not line up
- **Path mismatch**: the file is stored in a different place than expected

Use these results to decide what needs a rename, rescan, or cleanup.
